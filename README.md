Namma-Kathey 
Preserving Karnataka’s Heritage Through Technology

Namma-Kathey is an interactive Android storytelling app designed to introduce children to the inspiring heroes, freedom fighters, poets, and social reformers from different districts of Karnataka.

Instead of only knowing global superheroes, kids can now explore the real stories of local legends through fun storytelling, illustrations, quizzes, and interactive learning experiences.


Features
 Interactive Karnataka Map
Tap on any district to discover its local heroes.
  Swipe-Based Storybook Experience
Read illustrated stories in a smooth and engaging format using ViewPager2.
  Bilingual Support
Stories available in both Kannada and English.
  Text-To-Speech Integration
Kids can listen to stories aloud for better accessibility and engagement.
  Hero Quiz & Heritage Badges
Answer quiz questions and unlock achievement badges.
  Statue Finder

App Flow
Open the app
Select a district from the Karnataka map
Explore local heroes and stories
Swipe through illustrated story pages
Listen to stories using voice narration
Take quizzes and earn badges
Visit real-world memorial locations


Project Vision

Namma-Kathey aims to:

Build pride in regional history and culture
Help children learn values through real heroes
Promote Kannada language and local storytelling
Make history engaging through technology
